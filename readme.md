# Система управління бібліотекою (Library Management System)

### Опис проєкту
Цей проєкт є backend-додатком для автоматизації роботи бібліотеки. Він дозволяє керувати книгами, авторами, категоріями та відстежувати видачу книг читачам (loans).

**Склад команди:** 
Коханюк Анастасія

### Технологічний стек
* **Мова:** Java 17 (Spring Boot 3)
* **База даних:** PostgreSQL 15
* **ORM:** Spring Data JPA (Hibernate)
* **Контейнеризація:** Docker, Docker Compose
* **Тестування:** JUnit 5, AssertJ

### Інструкції з налаштування та запуску
1. Клонуйте репозиторій.
2. Переконайтеся, що Docker Desktop запущений.
3. Запустіть проєкт командою:
   ```bash
   docker-compose up --build
