#  Система управління бібліотекою (Library Management System)

##  Опис проєкту
**Library Management System** — це backend-додаток для автоматизації роботи бібліотеки.  
Система дозволяє керувати книгами, авторами та категоріями, а також відстежувати процес видачі книг читачам.

Проєкт реалізовано у вигляді REST API з використанням сучасного стеку Java-технологій.

1. Клонуйте репозиторій.
2. Переконайтеся, що Docker Desktop запущений.
3. Запустіть проєкт командою:
   ```bash
   docker-compose up --build

API буде доступне за адресою:
http://localhost:8080

### Основні сутності:
- **Author** — автор книги
- **Book** — книга
- **Category** — категорія книг
- **Reader** — читач бібліотеки
- **Loan** — факт видачі книги читачу

---

| HTTP Method | Endpoint       | Опис                         |
| ----------- | -------------- | ---------------------------- |
| GET         | `/api/authors` | Отримати список всіх авторів |
| POST        | `/api/authors` | Створити нового автора       |
| GET         | `/api/books/{id}`       | Отримати книгу за ID |
| PUT         | `/api/books/{id}/title` | Оновити назву книги  |
| GET         | `/api/categories` | Отримати всі категорії  |
| POST        | `/api/categories` | Створити нову категорію |
| POST        | `/api/readers`           | Створити читача                 |
| DELETE      | `/api/readers/{id}`      | Логічне видалення (soft delete) |
| DELETE      | `/api/readers/{id}/hard` | Фізичне видалення               |
| POST        | `/api/loans` | Видати книгу читачу |
